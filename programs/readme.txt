Selamat datang di TechOS versi 0.11
===================================


TechOS (Technology Operating System) merupakan sistem operasi berbasis 16-bit
berdasarkan sistem MikeOS dan TachyonOS yang dibuat oleh Faizal Hamzah, pemilik
The Firefox Foundation atas TechOS Developers pada tanggal 20 Desember 2016.

TechOS sudah free software dengan sifat Public Domains yang anda dapat ubah
kode sesuai dengan selera anda supaya TechOS bisa bekerja dengan baik. Oleh
karena itu saya mengharapkan pada anda, untuk membantu saya dalam pengembangan
TechOS agar dikenal oleh seluruh Indonesia, terlebih lagi dikenal seluruh dunia
meskipun ini berdasarkan MikeOS yang telah diakuisi dari lisensi.

Saya berterima kasih atas MikeOS dan TachyonOS telah menyumbangkan kode dengan
bermaksud untuk saya ubahkan ke dalam bahasa Indonesia, termasuk program
aplikasi BIN dan BAS (namun tidak banyak).

Akhirnya saya bisa merevisi TechOS, tetapi masih rilisan pengembangan alpha
dengan kode versi 0.11.

Apa yang baru tentang TechOS 0.11:

1.  Mendukung USB, Mouse, beberapa input.
2.  Menu sudah multitasking layaknya menu di sistem operasi lainnya.
3.  Perbaikan struktur GUI dan CLI.
4.  Menambahkan program-program BAS dan BIN dari sumber MikeOS dan
    TachyonOS.
5.  Mendukung restart/reboot dan shutdown pada GUI dan CLI.
6.  Tersedianya background.
7.  File Manager diganti oleh BlackSVK File Manager.
8.  Editor diganti oleh Yotta Text Editor.
9.  Masih banyak bug yang tidak bisa mengoperasikan menu all program,
    File Manager, File Viewer; Melakukan password dan membuat
    berkas.
